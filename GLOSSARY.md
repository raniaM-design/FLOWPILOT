# Glossaire de traduction PILOTYS

## Principes
- **Style** : SaaS professionnel, court, clair, actionnable
- **Registre** : Vouvoiement (formel, professionnel)
- **Tonalité** : Directe, efficace, sans fioritures

## Terminologie standardisée

### Concepts principaux
| FR | EN | Usage |
|---|---|-------|
| Réunion | Meeting | Toujours "Réunion" (singulier) / "Réunions" (pluriel) |
| Décision | Decision | Toujours "Décision" |
| Action | Action | Toujours "Action" |
| Projet | Project | Toujours "Projet" |
